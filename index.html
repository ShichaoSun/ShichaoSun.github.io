Research Experience

Math Word Problems Solving

Input a math word problem(insolving in addition, subtraction, multiplication, division, power), output the solution expression.

  • Implement the baseline (Seq2Seq model), and obtain better the result than the original paper(EMNLP 2017), as a result of more careful data processing, strict rules to restrict the generation and better hyperparameter setting.
  • Study the influence of different attention mechanisms containing Bahdanau attention, Luong attention, Dot attention, Minus attention, Bilinear attention. Design and implement multiway attention network motivated by considering different information from various attention mechanism.
  • According to the difference between machine translation with math word problem solving, design dynamic embeddings for number token at decoding side, the result is dramatically improved, approximately 2%.
  • Data Augmentation for solution expression: generating equivalent expressions to train the model by using commutative property and distributive property based on the motivation that makes the model to learn the math knowledge.
  • The code of dynamic embeddings and data augmentation is available from https://github.com/ShichaoSun/math_pade_da
  • Expression Tree can represent the relationship among the math units better than the sequence, so we design Seq2Tree model to Top-down generate Expression Tree for math word problems.
  • Code using PyTorch, one paper is under review by IJCAI 2019.
2017 - Present

Metaphor Detection

Classify whether a sentence contains the verb metaphor(like ''ignite passion'').

  • Detect verb metaphor by using Bi-LSTMs with word embeddings.
  • Study the influence of different granularity input(SVO, Dependency Sequence, Sentence).
  • Code with Lasagne(open source in: https://github.com/ShichaoSun/metaphor), and the paper is published in NLPCC 2017.

Chinese Word Embedding

Research Experience

Math Word Problems Solving

Input a math word problem(involving in addition, subtraction, multiplication, division, power), output the solution expression.

  • Considering the fact that mathematical meaning of the number is more important than its exact value, design dynamic embedding for numbers.
  • Propose a goal-driven tree-structured model to directly generate an ex- pression tree instead of generating solution expression, motivated by the human-like method of solving math word problems.
  • Source code is available at: https://github.com/ShichaoSun/math_seq2tree, and the paper is published in IJCAI 2019.
  • Data Augmentation for solution expression by generating equivalent ex- pressions, motivated by making the model learn math knowledge about commutative property.
2017 - 2019

Metaphor Detection

Classify whether a sentence contains the verb metaphor(like ''ignite passion'').

  • Model sentence by BiLSTM from word embedding to avoid using hand-crafted feature.
  • Design three different BiLSTM-based models motivated by three most common types of metaphor.
  • Propose an ensemble model of above three models.
  • Source code is available at: https://github.com/ShichaoSun/metaphor, and the paper is published in NLPCC 2017.
2016 - 2017

Chinese Word Embedding

Adopting word2vec algorithm to train Chinese Word Embedding and optimize it by constructing a better bag of words from dependency tree.

  • Source code is available at: https://github.com/ShichaoSun/DepWord2vec.
